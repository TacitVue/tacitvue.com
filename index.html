Atlas Mission Management System

Atlas Mission Management System supports the entire life cycle of any ISR Platform. The interface allows for seamless transition between the various ISR modes and supported platforms (e.g. airborne, ground, and UAS/UAV).

  •   Web-based application; no software installation needed
  •   Mission planning
    • Time of day
    • Sensor capabilities modelling
  •   Mission pre/post briefing playback
  •   Built-In-Test (BIT) support and reporting
  •   MIL-STD and NATO Symbology with customised support for clients
  •   2D/3D moving map
  •   Post-mission analysis
    • Metadata search
    • Geospatial regions
  •   Live video support
  •   VoIP, chat and file transfer support
  •   Able to run on embedded platforms with minimal size, weight and power
Ayn Camera Controller

Ayn Camera Controller provides platforms with the capability to control most industry standard electro-optical/ infrared systems (EO/IR) cameras and data link configurations.

  •   Web-based application; no software installation needed
  •   Digital and network video recorder (DVR/NVR) control
  •   Bookmarking for post-mission analysis
  •   National Imagery Transmission Format (NITF) clipping and annotation
  •   Directional ground station data link configuration
  •   Multi-user video streaming
  •   Moving map module
Imiter Sensor Simulator

Sensor availability for training, integration and support increases complexity and cost. Tacit Vue believes in simulation as a mechanism to fill this gap. With our adaptable simulator, engineers and trainers have the flexibility to use the entire platform in any locale.

Imiter can be integrated with Atlas or Ayn to support platform testing prior to delivery.

  •   Web-based application; no software installation needed
  •   Following generic sensor modules:
    • EO/IR
    • Data link
    • COMINT
    • ELINT
    • Synthetic aperture radar and ground moving target indicator (SAR/GMTI)
    • Chat/VoIP
  •   Vendor-specific modules can be developed/integrated as requested
